API - Overview

Items in a stream

If you want to get a list of items in your stream, you pass your organization's name (test-org in this example) in the request.

http://signalback.com/api/v0.2/item/search/?q=test-org

This response will be a list of objects.

JSONP

If you're building a JavaScript application that will run in a browser, you'll likely want to use JSONP. Just add the use the format and callback parameters in your request.

http://signalback.com/api/v0.2/item/search/?q=test-org&format=jsonp&callback=any-callback-name-want

This response will be a list of objects.
